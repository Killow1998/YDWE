# YDWE

[![Build status](https://ci.appveyor.com/api/projects/status/ybeps6jwp0nupxu6?svg=true)](https://ci.appveyor.com/project/actboy168/YDWE)

Based on Warcraft III World Editor, YDWE is a tool used for making Warcraft III game maps. 

## Compile

1. Visual Studio 2022 (17.0+) with C++20 support
2. Windows 10 SDK 10.0 or higher
3. Running `MSBuild YDWE.sln /t:Build /p:Configuration=Debug /p:Platform=Win32`
4. The compiled files are generated under the `Component\bin\` directory.
